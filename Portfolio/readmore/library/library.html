about Library Management System

Project in detail:

In this projects two modules are there-
1. Admin[Librarian]
2. Students

1. Admin (Librarian)

Librarian will login first.After login librarian see the operations like book operation and student operation. Librarian has access to update the student and book information. If librarian wants to change the password then he can change.

2. Student

After the registration of student he is authorised to issue and return book. After returning that books he can authorised to issue other books.

Technology used

Backend: Java,JPA with Hibernate
Database: mySQL

Worked on backend and database.
